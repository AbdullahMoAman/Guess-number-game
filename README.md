# Guess The Number 🎯

A simple game where you guess a number between **1 and 30**.

---

## How to Play

* Enter a number
* Click **Check**
* Follow the hint (Higher / Lower)
* You have **5 tries**

---

* DOM Manipulation
* Event Listeners
* Functions
* Conditional Logic
* setTimeout
